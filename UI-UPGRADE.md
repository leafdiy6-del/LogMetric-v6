# LogMetric Pro - UI 升级说明

## 🎨 设计风格: Soft Industrial + Glassmorphism

---

## ✨ 主要升级

### 1️⃣ **浅色/深色模式切换**
- 🌙 一键切换主题
- 🎨 浅色: Sand & Oak (沙米+橡木金)
- 🌑 深色: Deep Forest (深森林绿+橡木金)
- 💾 自动保存主题偏好

### 2️⃣ **Soft UI 设计**
- ✨ 新拟态阴影 (Neumorphism)
- 🔘 更大的圆角 (8-20px)
- 🌊 平滑过渡动画

### 3️⃣ **Glassmorphism 效果**
- 🔮 毛玻璃模糊背景
- 💫 半透明叠加层
- 🌟 微妙发光效果

### 4️⃣ **现代化配色**
```
浅色模式:
- 背景: #F5F3EE (沙米色)
- 强调: #B8956B (橡木金)
- 文字: #2C2824 (深灰)

深色模式:
- 背景: #0F1412 (深森林)
- 强调: #D4A373 (亮橡木金)
- 文字: #F0EBE5 (暖白)
```

### 5️⃣ **改进的组件**
- 🎯 按钮: 渐变+阴影+悬停动效
- 📝 输入框: 聚焦发光效果
- 📊 列表: 斑马纹+悬停滑动
- ⌨️ 专业键盘: 现代按键设计
- 🪟 弹窗: 滑入动画+玻璃背景

### 6️⃣ **微交互动画**
- ✨ 按钮点击缩放
- 🎭 列表行悬停滑动
- 🌊 弹窗滑入效果
- ⭐ 主题切换旋转

---

## 📁 修改的文件

| 文件 | 改动 |
|------|------|
| `index.html` | 引入新CSS，添加主题切换按钮 |
| `css/app.css` | 基础样式与深色主题变量 |
| `css/theme-addon.css` | 浅色/深色主题切换样式 |
| `js/app.js` | 添加主题管理函数 |

---

## 🚀 使用方法

### 切换主题
点击右上角的 🌙/☀️ 按钮即可切换浅色/深色模式

### 自动保存
主题偏好会自动保存到 localStorage，下次打开保持上次选择
